#LinkIt jQuery Plugin#

**LinkIt** is a simple jQuery plugin to attach a link to **any** element on a HTML page


##Version##
0.1.1

##Usage##
      $('element').linkIt({
        href: "http://www.jota.nl", // or whatever
        text: "Other text",  // or empty for actual text
        target: "_blank" // or "_self"
      });
